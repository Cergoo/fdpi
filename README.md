# fdpi
fuck dpi, http proxy

## install
```
git cone https://github.com/Cergoo/fdpi.git
cargo build --release
```

## run
```./fdpi -d2 -d4 -d7      // - for youtube.com, rutracker.org, https://index.minfin.com.ua/russian-invading/casualties/ 
google-chrome --proxy-server="http://127.0.0.1:8080"
```
see youtube
